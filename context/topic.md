# Topic

Type-safe prompt-template library with compile-time variable checking. Tagged-template literal API like `prompt\`Hello ${name}, your order ${id}\`` where the variable types are checked against an inferred or declared schema. Designed to make TS dev-experience for LLM prompts feel native — no string interpolation, no runtime surprises, types do real work.

## Args

- seeds: 3 (default)
- rounds: 2 (default)
- tribe: devtools (matt-pocock, addy-osmani, theo-browne, guillermo-rauch)
- web grounding: round-1 only (default)
